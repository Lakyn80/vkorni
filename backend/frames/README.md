# Frame Assets

## templates/
Place 10 PNG frame overlay images here, named `1.png` through `10.png`.
Images must be RGBA (with transparency) and will be resized to 800×1000 px.

Frame categories:
- 1–3: Decorative (artists, musicians, writers)
- 4–6: Classic museum (politicians, statesmen)
- 7–8: Minimal (scientists, academics)
- 9–10: Modern (athletes, sportspeople)

## fonts/
Place 10 TrueType font files here, named `1.ttf` through `10.ttf`.
Any .ttf fonts work (e.g. from Google Fonts).

If no font file is found, Pillow's built-in default font is used as fallback.
